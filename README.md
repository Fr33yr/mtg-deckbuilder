# Mtg-deckbuilder

## How it works 
It comsumes the Mtg api and uses firebase firestore for login and sotrage
to save your decks.

## How to run it in the local server
```bash
npm run dev
```

### required dependencies
- react
- redux
- react-router-dom
- swiper.js
- formik
- react-hot-toast
- firestore
